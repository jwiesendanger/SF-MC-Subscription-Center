%%[ /* UK Dynamic Branded Footer — Content Block Location: Parent BU > Content Builder > Content Blocks Usage: Insert this content block into the footer region of every UK email template. All values are AES-256 encrypted — subscriber key is never exposed in the URL. The child CloudPage uses AuthenticatedEmployeeID / QueryParameter to decrypt. */ SET @prefURL = CloudPagesURL(@childPageID, "action", "prefs") SET @unsubURL = CloudPagesURL(@childPageID, "action", "unsub") ]%%
